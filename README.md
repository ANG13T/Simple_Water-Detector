# Water/ Touch Detector

A simple circuit which detects water or touch by utilizing two 2n2222 transistors, an LED, and a piezoelectric speaker.

## Materials

- Breadboard
- 2 2n2222 transistors
- A power source
- Around 10 jumper wires
- Piezoelectric speaker
- 100 ohm resistor
- One LED
